# Run Ochoit Locally

## Requirements

- Node.js 22 or newer
- pnpm 10 or newer

## Quick Start

1. Install dependencies from the repo root:

```bash
pnpm install
```

2. Start the web app:

```bash
pnpm run dev
```

3. Open `http://localhost:3001`

## Notes

- The app will ask you to click `Start Audio` once before playback, preview, or recording work.
